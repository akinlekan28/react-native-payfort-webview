# react-native-payfort-webview

React native wrapper for payfort payment gateway

## Installation

```sh
npm install react-native-payfort-webview
```

## Usage

```js
import { multiply } from "react-native-payfort-webview";

// ...

const result = await multiply(3, 7);
```
